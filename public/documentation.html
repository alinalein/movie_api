Documention for the Movie_API


The API should retrieve the URL from the client , the request of the client and return a corresponding response by the server. The server will fetch the data from a database.



Business Logic URL HTTP Method Query parameter Request body data format Response body data format
Return a list of ALL movies to the user /movies GET https://movie-api-lina-834bc70d6952.herokuapp.com/movies None A JSON object holding data about all movies with this format: 
                            {
                                Title: {type: String, required: true},
                                Description: {type: String, required: true},
                                Genre: {
                                    Name: String,
                                    Description: String
                                },
                                Director: {
                                    Name: String,
                                    Bio: String
                                },
                                Actors: [String],
                                ImagePath: String,
                                Featured: Boolean
                            },
                            {...}
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user /movies/title/[Title] GET https://movie-api-lina-834bc70d6952.herokuapp.com/movies/title/:Title None A JSON object holding data about a specific movie: 
                            {
                                Title: {type: String, required: true},
                                Description: {type: String, required: true},
                                Genre: {
                                    Name: String,
                                    Description: String
                                },
                                Director: {
                                    Name: String,
                                    Bio: String
                                },
                                Actors: [String],
                                ImagePath: String,
                                Featured: Boolean
                            }
Return data about a genre (description) by name/title (e.g., “Thriller”) /movies/genre/[Genre] GET https://movie-api-lina-834bc70d6952.herokuapp.com/movies/genre/:Genre None A JSON object holding data about a specific gerne: 
                            {
                                Name: String,
                                Description: String
                            }
Return data about a director (bio, birth year, death year) by name /movies/director/[Director] GET https://movie-api-lina-834bc70d6952.herokuapp.com/movies/director/:Director None A JSON object holding data about a specific director 
{
                                Name: String,
                                Bio: String,
                                Birth: String
                            }
Allow new users to register /users/register POST https://movie-api-lina-834bc70d6952.herokuapp.com/users/register A JSON object about the user should be send, including the following data: 
                        {
                            Username: {type: String, required: true},
                            Password: {type: String, required: true},
                            Email: {type: String, required: true},
                            Birthday: Date
                        }
 A JSON object holding data about the user that registered
Allow users to update their Username /users/update/[Username] PUT https://movie-api-lina-834bc70d6952.herokuapp.com/users/update/:Username A JSON object about the data of the user that is supposed to be updated: 
                        {
                            Username: {type: String, required: true}
                        }
 JSON object with the updated info about the user
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added—more on this later) /users/[Username]/movies/add/[MovieID] PUT https://movie-api-lina-834bc70d6952.herokuapp.com/users/:Username/movies/add/:MovieID None JSON object with the updated info about the user including the favorite Movies
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later) /users/[Username]/movies/remove/[MovieId] DELETE https://movie-api-lina-834bc70d6952.herokuapp.com/users/:Username/movies/remove/:MovieID None JSON object with the updated info about the user including the favorite Movies
Allow existing users to deregister (showing only a text that a user email has been removed—more on this later) /users/deregister/[Username] DELETE https://movie-api-lina-834bc70d6952.herokuapp.com/:Username None A text message that informs the user if the profile was successfully deleted
Allow existing users to log in to own profile with Username and Password as Params /login POST https://movie-api-lina-834bc70d6952.herokuapp.com/login None JSON object showing the users details and his token
